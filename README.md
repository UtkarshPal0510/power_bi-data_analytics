# Power BI Data Analytics Project

This repository showcases my Power BI and data analytics skills using a collection of job postings data from 2024. 

---

## 📊 Project Overview
The goal of this project is to analyze monthly job postings, extract meaningful insights regarding hiring trends, and present them through an interactive Power BI dashboard.

## 📁 Repository Structure
* **`dashboard.pbix`**: The core Power BI file containing the data model, transformations, and interactive visualizations.
* **`job_postings_01-2024.xlsx` to `job_postings_09-2024.xlsx`**: Monthly Excel datasets (January – September 2024) serving as the data source for this analysis.

## 🛠️ Key Skills & Features Demonstrated
* **Data ETL:** Importing, cleaning, and appending multiple monthly Excel files using Power Query.
* **Data Modeling:** Creating relationships between tables and establishing a star schema where applicable.
* **DAX Formulas:** Writing measures and calculated columns for advanced metrics (e.g., MoM growth, total postings).
* **Data Visualization:** Designing a clean, intuitive, and user-friendly dashboard layout.

---

## 🚀 How to View the Project
1. Clone this repository or download the `dashboard.pbix` file.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
